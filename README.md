# Proyecto 2

## Sistema de Particulas

¡Hola de nuevo!
Este es mi segundo proyecto y exámen de interciclo, en el cúal he creado un sistema de particulas que tienen un tiempo de vida establecido y además nacen a partir de la posición del cursor en la pantalla.
El código está pensado para que visualmente se pueda observar una figura parecida al fuego, siempre y cuando el cursor se mantenga en reposo. Además, **cuando el cursor esté presionado ocurre una dispersión mayor de las partículas, simulando una especie de explosión o ráfaga de fuego**, como si hubiera una fuente de viento o una explosión que afecta el comportamiento de las partículas y brillos generados por el sistema.
¡Espero que puedan disfrutar de lo que he creado!
